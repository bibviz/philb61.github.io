Generated Output ---------------- If you would like to contribute your translation, please [open a pull request or issue on Github](https://github.com/danielgtaylor/bibviz). Make sure to include your generated translation if you open an issue, as well as the language it is written in. If you would like to load an existing translation, they can be found in the [locales directory on Github](https://github.com/danielgtaylor/bibviz/tree/master/web/locales).
